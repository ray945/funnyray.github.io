发表于 2014-08-27 更新于 2020-03-23 分类于 学生时代的瞎BB

我是一看电视剧就心塞星人。

昨天在网上看到了一部电视剧，叫《匆匆那年》，是根据小说改编的，同名的电影十月也会上映。我一向对这种怀旧向的标题毫无抵抗力，看我之前的日志就知道，因为我就是一个喜欢怀旧的人。

不要在意“那些”细节

发表于 2014-08-12 更新于 2020-03-23 分类于 学生时代的瞎BB

记得小学毕业时填写同学录，总有一项是“口头禅”，那时候似乎也没什么真正的口头禅，所以就很装b地写一句格言什么的，其实自己从来都没用过那句话……

嘿嘿，不要在意那些细节～

如果现在让我们来写自己的口头禅的话，估计也是一年变一个样。在网络待得久了，总是不知不觉地就把一些网络流行语挂在了嘴边。比如“不要迷恋哥，哥只是个传说”、“神马都是浮云”之类的流行语曾一度成为全民口头禅。

什么？你没把这些话当做口头禅？算了，不要在意那些细节～

反正最近我跟周围的人说的最多的一句话就是……诶，刚刚好像说过了？嗯，不要在意那些细节～

阅读那些事儿

发表于 2014-05-17 更新于 2020-03-23 分类于 学生时代的瞎BB

最近的生活一直处于一种很规律的状态。白天就上上课，然后看看android的视频，晚上小小的放松一下，然后十点过回到寝室，不带电脑，就一个人，一个手机，一副耳机。

之所以最近不带电脑回寝室，一是觉得从办公室到寝室搬来搬去太麻烦，二是因为最近重新喜欢上读书了。

用重新二字，是因为小时候的我其实是个很喜欢看书的人。很小的时候家里有整整一柜子的小人书连环画，是我爸小时候买的，很有历史了，不过那些书真的很好看，我相信所有70,80后都会对那种连环画很怀念，因为那可能是很多人养成读书习惯的起源。很庆幸老爸把这些小人书保存得很好，让我小时候有着别样的乐趣。后来，知道了一个叫新华书店的地方，每天，新华书店里都会坐着十几个小朋友，就坐在地上，聚精会神的看着各种书，漫画、童话、或者小说……最开始的时候所有的书都是可以随便让你翻阅的，可是到了后来，店家觉得这样似乎很影响生意，所以后来很多书都不允许小朋友在那儿看了。不过这依旧阻止不了那些小书虫，我们依旧每天跑到书店去看书。
