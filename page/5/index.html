阅读那些事儿

发表于 2014-05-17 更新于 2020-03-23 分类于 学生时代的瞎BB

最近的生活一直处于一种很规律的状态。白天就上上课，然后看看android的视频，晚上小小的放松一下，然后十点过回到寝室，不带电脑，就一个人，一个手机，一副耳机。

之所以最近不带电脑回寝室，一是觉得从办公室到寝室搬来搬去太麻烦，二是因为最近重新喜欢上读书了。

用重新二字，是因为小时候的我其实是个很喜欢看书的人。很小的时候家里有整整一柜子的小人书连环画，是我爸小时候买的，很有历史了，不过那些书真的很好看，我相信所有70,80后都会对那种连环画很怀念，因为那可能是很多人养成读书习惯的起源。很庆幸老爸把这些小人书保存得很好，让我小时候有着别样的乐趣。后来，知道了一个叫新华书店的地方，每天，新华书店里都会坐着十几个小朋友，就坐在地上，聚精会神的看着各种书，漫画、童话、或者小说……最开始的时候所有的书都是可以随便让你翻阅的，可是到了后来，店家觉得这样似乎很影响生意，所以后来很多书都不允许小朋友在那儿看了。不过这依旧阻止不了那些小书虫，我们依旧每天跑到书店去看书。

发表于 2014-02-20 更新于 2020-03-23 分类于 学生时代的瞎BB

寒假已经玩儿了将近两个月了，马上也要开学了，该看的没看，该学的没学，连该玩儿的都没有玩儿得尽兴……最近总爱发呆，就像老年痴呆了一样，幻想着各种美好的未来——工作、女友、家、生活……

外国的月亮有多圆？——我的国产情结

发表于 2013-12-27 更新于 2020-03-23 分类于 学生时代的瞎BB

12月24日，我大天朝地区苹果价格疯涨，但天朝人民就跟不知道似的，依旧疯狂购买，然后，再买点包装纸，把苹果一个一个包装起来，送给周围的亲朋好友，还得说上一句：“平安夜快乐~”

平安夜，即圣诞前夕，是千千万万的欧美人风尘仆仆地赶回家中团聚的日子。类似中国的除夕吧“平安夜嘛，自然要吃平安果”。。。我不知道这个风俗是谁兴起的，说不定是起源于哪个无良水果商贩的营销策略，但我敢肯定的是，这个风俗绝对不是从西方传过来的啊！平安（Silent）和苹果（Apple）绝对不是谐音啊！于是，平安果，只是一种中国人以自己的方式莫名其妙的过着外国人的宗教洋节日的一种土方式。但即使这样，又有什么关系呢没有人会在意这点。一群小青年冲上广场，拿着充气棒，乱敲一气，乐此不疲地欢度着这样一个本不属于中国人的节日。
