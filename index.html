AWS IoT KeepAlive 心跳流程

发表于 2019-06-15 更新于 2020-03-23 分类于 经验总结

首先简单说一下，AWS Iot SDK 的 mqtt 模块是基于 paho 标准的 mqtt client java 进行开发的，没有加入 paho 的 android 扩展包，所以心跳只能通过 TimerPingSender 进行定时发送。但由于在 android 系统中，如果系统长时间没有 cpu 活动，会进入休眠状态，而简单的 Timer 是无法唤醒 cpu 的，这就导致官方的 AWS IoT SDK 无法理想地运行在游乐园项目（相机需一直保持 mqtt 在线）。所以游乐园项目使用的 AWS IoT SDK 是我们手动加入了android 扩展包中的 AlarmPingSender 后自己编译的。SDK 源码目录：goldsand/packages/apps/aws-android-sdk-iot-source/，直接通过 gradle 编译。

幻想，梦想

发表于 2015-08-01 更新于 2017-01-01

又是很长一段时间没上博客了，前段时间忙着期末、实训、课程设计，以及找实习工作，博客没工夫上，Ray天气也没工夫更新……好在实习很顺利地找到了。

已经整整实习两周了，感觉非常好。

实习的单位是一家中小型的创业型公司，有自己的app产品，而我就在Android团队做开发，做的东西很有意思。团队成员都很年轻，基本都是85后、90后，很有意思，也很有活力，相处起来很轻松。朝九晚五，中午在公司里用微波炉加热便当吃，周末双休，不强制加班，各种福利一应俱全，工资待遇也比较满意。

我在优客逸家上找了间房，套三，三个人合租。装修很漂亮，宜家风格，很有格调，每天下班回去都感觉心情舒畅，离公司也很近，步行十几分钟就能到公司。室友人也不错，做的一手好饭。每天晚上下班到家，做好晚饭，以及第二天中午的便当。刷刷微博，看看电视，差不多就睡了。不算小资，但很惬意……

是不是很羡慕这样的生活？其实我也很羡慕……因为这特么全都是我之前脑补过无数次的幻想，幻想好了实习后的每一天，每一刻，每一个细节，甚至想好了每天要做什么菜，看什么书，玩什么游戏，约什么妹子【这条划掉】……

然而现实的画风好像跟我想象的不太一样……真实情况是这样的：

已经整整实习两周了，感觉不怎么好。

实习的单位叫“成都XX科技有限公司”，是深圳总部在成都的一个研发中心，主要为一些出口国外的手机和智能设备做软件支持，我被分到了Framework组，做一些Android系统级的应用，比如电话、短信、设置、启动器……总之就是系统内置的那些apk。然而在这之前，我只接触过Android最顶层的应用开发，也想着做这一块的工作，却没想到一不留神，就往底层靠近了一部，开始研究应用框架层的东西，这就导致我之前学了那么久的东西，变成了一点小小的基础，而更多的东西，需要从头学过。刚进公司有三个月的培训期，而这两周，我都在看各种各样的文档资料电子书，Linux，Makefile，Android Framework，Git/Repo/Gerrit/Bugzilla开发流程……各种胡吃海塞，一股脑地全往脑子里装，没实际上手，每天看起来闲得不行，然而脑子却各种难受。公司还算比较大，光算成都这边，就有300多号人，在软件园的一栋大楼里占有两层，实在霸气。公司大了，自然条条款款就比较多了，刚进来的时候，各种会议、考核、组内分享搞得我一头雾水……好在小组里的大哥大姐们人都挺好的，男的幽默，女的豪爽，没个几天也就混熟了。朝九晚六，中午在园区餐厅里吃，味道还不错，就是小贵小贵的。不强制加班，但是加班有调休，有零食，加到九点以后还有餐补，周末双休，平时还有各种活动，比赛、旅游、聚餐什么的……虽然比我想象的苦逼了点，但公司的福利什么的，还是挺让人心动的。

公司的福利不错还体现在给实习生提供免费住宿，这对刚刚参加实习，工资刚刚够养活自己的我来说，简直不能再棒。宿舍是园区里的一栋廉租公寓，和大学里差不多的四人间，有空调，有热水器。室友都是公司同事，也是在实习或者刚毕业的，甚至有一个还是我大学同学。宿舍下楼走两步就到公司了，中午下班再走两步就到园区餐厅了，依旧三点一线，从校园到软件园，好像没什么变化，而这一个月就连白天做的事也和在学校时差不多，没别的，就是学。晚上休息一下，在网上闲逛一会儿，一天也就过去了。不算惬意，但还凑合……

本想着工作了，就打死都不再找家里要钱了，但现实着实有点残酷，中午在餐厅一不小心就能刷掉18块钱，这对培训期间只有2k生活补助的我来说，简直太过奢侈。不得已，连续两个周末都跑回家待着，毕竟车费只要几块钱，两天的饭钱却有好几十……因为是中途入职，这个月估计也就拿得到一半，接下来的这个月能不能喂饱自己也着实是个未知数，宽带也还没办，每天可怜地蹭着隔壁wifi，一到晚上就卡的不行【隔壁的哥们儿真对不起】，想想目前手头的票子，也只能忍忍了。

不过吧，我还是挺乐观的，想想也就前三个月苦点儿，等熬过这段培训期，就能慢慢步入正轨了~

曾经的幻想在这两周的时间里，变成了一个小小的梦想。这个梦想其实也不远，很近，与现实也就隔着一步之遥，这一步，叫做努力。等过上曾经幻想的惬意小生活时，再来冲刺下一个梦想吧，比如改变世界什么的……

不过我得先想办法用50块钱熬到5号发工资……

最后一个寒假

发表于 2015-01-29 更新于 2020-03-22

今天是2015年1月29日，距离放假已经过去整整20天了。

很难说清楚这20天的假期里我究竟做了些什么事，因为连我自己都似乎得了健忘症，把这些天的事情忘了一大半，只记得很少的一些事。

起床，吃饭，上网，吃饭，上网，吃饭，上网，吃饭，睡觉……

这大概是我忘记的那部分，历史总是惊人地相似，每次假期都过着这样山中不知日月的生活。不过还好，我还没到忘记时间的地步，毕竟随意瞟一眼右下角就能知道。

事实上，即便是在学校，我的生活也基本上处于这样的“规律”之中。如果我能记得住过去10天之内发生了什么，那一定是发生了什么变化。我讨厌变化，打乱了我规划已久的安排，比如放假前得知因为自己的小聪明，而导致选修课学分不够的噩耗……我又渴望一些变化，让生活不是那么的无趣，比如放假后买了两个新东西。

而这20天里我能记住的那些事，全部得益于这两个小东西……

买手机是盼了很久的事情，也紧巴巴地攒了很久的生活费，本想换个好用的手机，不root，不刷机，老老实实地使用，更好地服务生活。可我终究是个爱折腾的人，入手之后还是抵不过手贱……手机到手后，花了很多天的时间刷机，疯狂地刷，当初选择一加可能也是因为潜意识里看上了它适配性强的特点。选包、备份、刷机、还原、试用……这样下来基本上就是一天。短短的几天时间，我刷了不下10次，始终找不到一个让我称心的ROM，最后还是换回了官方包。

手机的系统终于定下来之后，就开始折腾软件了，64G的ROM，3G的RAM，让我开始任性地下载各种APP。最夸张的时候，在家玩儿了整整两天的配音秀，对自己的声音还颇为满意，差点被自己的配音感动到……

那几天就这样每天呆在家里，穿梭于电脑和手机之间，渐渐地，新手机也彻底融入了生活，变成了“规律”的一部分。然后，我毫不犹豫地买了另一个小玩意儿。

如果说手机给了我很多宅在家里的理由，那么新买的小米手环重要给了我一个出门的理由，并且十分充分。

智能手环已经不是什么新鲜玩意儿了，很早就有厂商在做，只不过一直都是有钱人的新鲜玩意儿，跟我没半点儿关系。直到伟大的雷布斯开始涉猎智能设备领域，让我们这些屌丝有了体验科技的机会。

在小米手环上市之初，就想买一个体验一下，奈何手机太烂，系统不支持，只得眼巴巴地望着周围的人抢先体验。其实大多数买手环的人并不是真的出于运动的需要，更多的还是好奇。所以在我入手的时候，周围买了手环的朋友，早已把它们扔在了一边。

我也不是个爱运动的人，所以本身也没对督促运动这块儿功能抱什么希望。结果戴上手环的那周，我居然在外面晃悠了好几天，一点儿都没在家闲着。可能是因为确实在家宅太久了，想出门转转，其次是因为，每天看着自己走了上万步的感觉真的超级爽。

其实，不用想也知道，对这个长达两个月的寒假，我是做好了充分的规划的，并且绝不是这样。放假前，我在协会13级的会员群里说：

亲们，寒假到了~这意味着什么呢？可以愉快地玩耍了吗？NO！！！寒假最适合学习了！！！
之后的事你们都知道了，毫不意外地发生了意外。我的计划永远赶不上我的变化，没看完的书，没写完的代码，都在我扑向沙发的那一刻被抛到九霄云外。直到几天前，收到邮件，一个博友留言提醒我已经两周没写东西，才想起这茬儿……更纠结的是，已经持续了半年多的说说签到，也断了好几天……

按照往常的尿性，寒假到这个时候就该开始在内心深处痛斥自己对时间的挥霍了，这一次却也懒得后悔了，毕竟是学生时代的最后一次寒假~

不过也该按计划执行了，今天总算在难得的阳光下，静下来看了一天书，算是一个好的开始吧，毕竟是学生时代的最后一次寒假……

2014，就这样吧

发表于 2014-12-30 更新于 2017-01-01

还有一天，2014就又要翻篇了。

突然发现，自己貌似越长大，就越对跨年这件事儿没感觉了。高中的时候，提前半个月就能开始兴奋，跟朋友讨论某某卫士跨年晚会请了谁谁谁之类的。上了大学，跨年基本上都是在宿舍和室友们一起的，想看个晚会吧，网络直播卡得人想哭，到点后也就我一个人在那儿激动，别人该LOL还继续LOL，久而久之，我也就没了兴致。本来连博文都懒得写了，但出于强迫症，还是憋着自己写点什么，好证明我又多活了一年。最近临近放假，成天玩儿，感觉脑子里都是浆糊，所以就不写什么深刻的反思了，随随便便回顾一下我的2014吧。

2014，我换了个微博。要抛弃以前那个1000+粉丝的微博真的还是挺纠结的一件事，不仅仅是换个账号那么简单，更重要的是抛弃了微博上的一群游戏圈的朋友，也抛弃了那个每天几十条微博，且无节操无下限的自己。

2014，我改了个ID。实不相瞒，之前用了几年的“Ray沧海剑覆”这个网名就是为了装逼而生，别人问我啥意思，我都说不出个所以。所以在换了微博后，思前想后，然后郑重地把所有的社交ID全部换成了“Ray不忘初心”。不忘初心是一个早就被文艺青年玩坏了的词，但对我自己来说，确实有着很好的警示作用，无数次的偏离轨道之后，猛然意识到，自己或许应该重拾曾经的自己。顺便也给自己树立了个人生信条：专注、极致、不忘初心。

2014，我买了个Kindle PaperWhite 2。这也是我第一次毫无顾忌地根据自己的意愿去买一个东西。家里条件挺普通，零花钱实在没有多少，自己又总是太懒，不愿去做一份兼职，本来说攒钱买手机，结果攒了很久也没攒起来。在周围某些大牛推荐了Kindle，自己也意识到确实该多读点书后，二话不说就拿着生活费买了一台……没跟父母汇报，也没考虑接下来怎么吃饭。后来那个月的生活简直可怜，每天勒紧裤腰带，想到就心酸，好在“美团”和“饿了么”抢外卖生意，每天的外卖便宜到死，才没让我在那个月饿死。

2014，我没放弃这个博客。去年年底开这个博客的时候，所有人包括我都觉得自己是图个新鲜，玩儿几个月就会扔了。结果令我自己也挺意外，虽然每个月的文章数量太少，而且质量参差不齐，但至少坚持下来了，还认识了一堆的文艺青年和技术大牛。虽然到现在为止，博客基本依旧上没什么流量，但自己也倒是乐在其中。离开QQ空间里嘲杂的环境，这个博客倒也成了我的一,方乐土。

2014，我认识了简书和知乎。身处在一群牛人之中，这几乎就是一件必然的事情。我开始试着把自己的想法和文章放到上面，去接受更多人的鼓励和批评。

2014，我勉勉强强半只脚跨进了科技圈。成天在网上看科技新闻，看电子产品测评，在无意识的情况下，倒也了解到了不少的东西。甚至 有想过自己以后也去做手机测评之类的工作，比起没日没夜地写代码，把玩这些东西似乎更能激起我的热情。

2014，我自学了吹箫。虽然早就想学一门乐器，但总是觉得出去找老师太贵，自学又太麻烦。今年总算在一阵激动过后买了一根非常便宜的箫，并且摸索着学了好一阵，成果的话，不佳，但在状态好的情况下，能比较好地吹奏“飞雪玉花”和“回梦游仙”两首曲子。

2014，我成了学校网络协会的会长。这本应该是一件值得自豪的事，但自己实在说不出口。协会人不多，但即便在这少数人里，我也自知水平实在不够。Android学了一年多了，可依旧停留在入门阶段。若我自己来说，是万万配不上这“会长”一职的，好在自己办事还算靠谱，勉勉强强带着协会过了一学期。技术方面，也在抓紧提升，毕竟明年6月就要开始找实习了。

2014，我发现周围的人开始逐渐拉开差距。有些人在成为大神的路上越走越远，有些人则在虚拟世界中越陷越深。我曾以为来了一个平庸的学校，周围的人都注定了和我一样平庸，结果发现是自己太过鼠目寸光。同一届的学生里，有人挣了钱，有人收到了offer，还有人受到了投资方的关注，而我还在没头没脑地看着《XX入门实战》，写着无聊的小程序。这一年，第一次看到人和人之间的差距如此巨大，第一次感到如此心慌。

2014，我同以往一样，干着不能再普通的事，做着不能再普通的人。

明天早上启程去广元，表哥要在元旦那天结婚，他在新年伊始实现了巨大的转变。那我呢？算了，没什么新年愿望，只希望身体健康，一切都好……

2014，就这样吧……

“墙”外的世界

发表于 2014-12-07 更新于 2017-01-01

前两天上课的时候，不知道为什么，突然有种强烈的想“翻墙”的冲动……

之前用Windows系统的时候，一直是用GoAgent或者××门软件来翻墙的，速度很慢，不过也还勉强能用用。可是换了Ubuntu单系统后，××门用不了，GoAgent配置也不方便，得先上Google，这是个死循环，然后也就懒得弄了。

不过对一个在学习Android开发的学生来说，不能翻墙确实是件糟心的事。不论是上Android官网，还是下载SDK，以及Google Play服务之类的，都需要跨过那道高墙。想起之前一些大牛推荐过的一款VPN，就跟朋友搭伙买了一年的。（云梯VPN，15/月，每月100G流量，不看视频、下载大文件什么的话，两个人完全够用，支持PPTP，L2TP两种连接方式，电脑和手机各个系统平台都能用，这和GA比起来算一个优势吧，也还比较稳定。点此链接购买的话你和我都能获得10块钱的优惠→_→）

VPN帮我在高墙上开了一个小门，虽然不大，还得收过路费，但总算能让我愉快地走出去看看了。整整两天，我在寝室里注册了各种我能想到的知名社交网络，Twitter，Facebook，Instagram……然后上Google把每一个功能都点进去尝试了一下。我本以为虽然Google被墙，但百度之类的国内网站功能也基本能替代之，结果却发现完全不是这样。之前从来没想到去体验一下Google的各种服务，而仅仅这两天的浅尝，却让我感觉就像发现了新大陆一样。

除了Google本身的各种完善的服务以外，Chrome商店里的各种扩展应用和插件简直强大到令人发指。其跨平台性补充了Ubuntu在应用软件上的硬伤，同时，结合Android手机上的Google Play服务以及一些同样需要翻墙才能使用的第三方应用，几乎可以做到媲美苹果那样，平台之间的无缝衔接。

为了在手机上也能体会到墙外的乐趣，我特意将手机系统刷成了AOSP（Android Open-Source Project，也就是最原生的Android系统），然后刷入了全套的Google服务，并配置好了VPN。用了两天后，发觉那是和在电脑端同样出色的服务体验，甚至开始怀疑我究竟有没有用过Android手机……毕竟国内手机厂商几乎全部删除了Google的服务框架，换成了自己的。即便在Google还未退出中国的几年前，也没有几家手机厂商内置了完整的Google服务，这让我感觉到了深深的遗憾。国产手机厂商说：原生Android自带的Google服务并不适合中国人使用，于是换成了他们自己的“适合”国人自己的各种服务。于是，天朝人民再不能体会到Google Play商店里各种好用、美观的优秀APP，取而代之的是遍地开花的各种应用商店，和在其中疯狂生长的良莠不齐、充满广告、优化差劲的国产应用。

你说国产应用商店是免费的？确实是这样没错，不过我并不认为现在的人们会舍不得那一点点的钱……比如我。从几年前开始，我开始学着为了各种正版体验买单：花钱买正版游戏，花钱在亚马逊上买正版电子书看，花钱在网上看正版视频，最近又在Ubuntu系统上花钱用Spotify听正版音乐……曾在科忠的博客（YesBlog）中看到这样一句话：这是个体验经济的时代。比起在应用中看到各种额外收费或者丑陋无比的广告栏，我更愿意在获取应用的时候支付一小笔费用。而且，看正版书、听正版音乐、看正版电影，也许是心理作用，但确实能给人一种更加愉悦的心情——每个人的付出都应得到回报，每个人都应为优秀的体验买单。更何况比起生活中一些乱七八糟的花销，为优秀的正版体验买单只是一笔小到可以忽视的费用。

但真正可怕的是，我们想体验正版，想为优秀的作品买单，但没有机会……我们被束缚在高墙之内，听旁人诉说着外面世界的精彩，却无缘亲自去探索一番外面的花花世界。多年以前，我们周围是没有这面墙的，我们本可以和全世界的人民一起在Facebook交友，在Twitter分享，那才是一个真正实现全球互联的世界，那样的“互联网”才能名符其实。“全世界都在用Twitter，只有我们用微博。”这句话听起来真的挺苦逼的……

确实，人言可畏，言论自由是可怕的……外面的确充斥了很多的我们不能、不想、不应该听到的声音，但我觉得这并不应该成为一个问题，这个问题有千千万万种解，而他们偏偏选择了围追堵截这最烂的一招。和全中国大多数的父母一样，我们的祖国母亲也选择用她独特的方式呵护着我们。她生怕我们在外面学坏，甚至担心我们看一部《生活大爆炸》或者《破产姐妹》之后，培养了多年三观就歪了……而我们的行动也不负众望，和全天下大多数孩子一样，父母越不让你怎么样，就越想怎么样……我只想说，我今年20，已经成年了。

断掉VPN，我又回到了高墙之内，外面的一切就像一场梦，那么真实，却又无法触摸。手机里刚刚还让我惊叹的应用，瞬间也变成了废物一个……

不过还好，墙上的那扇门还在，无聊的时候我依然能悄悄溜出去逛逛……虽然不太方便。

这一夜，我听见、看见的美好

发表于 2014-12-02 更新于 2017-01-01

这是一个安静的夜晚。

窗外并没有下雨，

但我听见了……

雨水打在屋顶，然后顺着屋檐，滴落到地面的水洼，

每一滴都听得如此清晰。

突然觉得很神奇，原来那个曾经被自己骂过无数次的“破学校”，竟然有着这样的魔力……我还在学校的时候，她将自己的美隐藏起来，等我走了，她就开始讲述一个个美妙的童话故事了。

也许她是在惩罚我，因为我的不懂欣赏，因为我六年来无情的挥霍……我曾无数次地踩着满地的落叶走过，却从未停下脚步……

这些景物在我离开学校的两年内并无太大变化，一切都还是原来的样子，但又好像不太一样。

一些大学的朋友在空间里看到我高中的照片，都会说：“你们高中好漂亮，真幸福！”

对着那些几年前的照片，我说：“是啊，真漂亮。”这句简单的话语中，多过自豪的，是深深的遗憾。

原来，我们也曾拥有过这样的金色校园啊……

原来，我也曾生活在那样一个童话的世界……

可惜，那些年的美好，被那些年的我们，一一错过。

不过还好，在这个夜晚，我终于把我遗失的东西找回来了。

也许今后，我会扎根成都，结婚生子，

然后把自己的孩子送回那个校园，让他去那些我曾走过的地方。

他会在那片盛开的樱花树下，或是在漫天飞舞的银杏叶中，

牵起一个人的手，完成我未完成的梦。
