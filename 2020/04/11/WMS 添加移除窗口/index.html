WMS 添加/移除窗口

发表于 2020-04-11 分类于 学习笔记

添加窗口

客户端添加窗口需要通过 IWindowManager.addToDisplay() 或 IWindowManager.addToDisplayWithoutInputChannel() 这两个方法，这两个入口本质上都是调用了 WindowManagerService 的 addWindow() 方法

Session.addToDisplay() / Session.addToDisplayWithoutInputChannel()

WindowManagerService.addWindow()

流程小结

  1. 获取或创建窗口要添加的 DisplayContent

  2. 从 mTokenMap 取出已注册的 WindowToken，没有的话隐式创建一个

  3. DisplayPolicy 调整 LayoutParams 的一些成员的取值

  4. WMS为要添加的窗口创建一个WindowState对象，创建过程中根据窗口类型初始化 BaseLayer，SubLayer

  5. DisplayPolicy 为添加窗口做一些准备

    • 确保 STATUS_BAR,NAVIGATION_BAR 为单例

    • 需要返回 ADD_OKAY 后才能添加窗口

  6. 打开 InputChannel 并注册到 IMS

  7. 通知 IWindowSession 窗口添加，准备 SurfaceSession

  8. 将 WindowState 添加到 mWindowMap 中

  9. 按顺序将 WindowState 添加到 WindowToken

  10. 如果 Window Type 为的话，设置/更新IMEI target

  11. 更新 FocusedWindow，FocusedWindow 改变的话通知 IMS

  12. 为子窗口调整Z-Order

  13. 需要的话设置/更新IMEI target

移除窗口

WindowManagerService.removeWindow()

WindowState.removeIfPossible()

WindowState.removeImmediately()

流程小结

  1. 获取 Window 对应的 WindowState

  2. 销毁 InputChannel

  3. 动画在运行的话就推迟移除窗口

  4. 开始移除窗口

  5. 更新 IME Target 窗口

  6. 通知 DisplayPolicy 移除 window

  7. 将对应的 Session 从 mSessions 中删除并清除 Session 对应的 SurfaceSession 资源

  8. 做一些清理工作，从各种map,list中移除

  9. 更新 Focused Window 和 IME Window
